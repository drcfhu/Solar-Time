Napóra v3.0 – GitHub Pages / iPhone PWA

Telepítés GitHubra:
1. Csomagold ki ezt a ZIP-et.
2. A benne lévő fájlokat töltsd fel ugyanabba a GitHub repositoryba, ahol a régi Napóra van.
3. A régi index.html, app.js, style.css, manifest.json és service-worker.js legyen felülírva.
4. Commit: "Update Napóra to v3.0".
5. Várj 1–2 percet, majd nyisd meg a meglévő főképernyő ikonnal.

Ha iPhone-on még a régi verzió jelenik meg:
- Nyisd meg egyszer Safariban az oldalt, húzd le frissítéshez.
- Zárd be a Napóra ikont a háttérből, majd nyisd meg újra.
- Végső esetben töröld a főképernyő ikont és add hozzá újra.

Új funkciók:
- HU / IT / EN felület.
- Valódi helyi napidő GPS alapján.
- Napmagasság, azimut, napkelte, valódi dél, napnyugta.
- Holdfázis.
- Mozgó napóra-árnyék.
- Nap-sörszög napnyugta előtt egy órával.
- Napforduló és napéjegyenlőség üzenetek.
- Automatikus cache-frissítés service workerrel.
- Offline működés.
